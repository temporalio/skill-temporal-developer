# Cadence Development Skill

A Cadence-focused development skill for building and debugging applications with the official Cadence surface area.

This repository now targets:

- Cadence core concepts
- Cadence Go SDK
- Cadence Java SDK

It intentionally does not try to preserve Temporal-only SDK and platform guidance.

## Scope

This skill is designed for work involving:

- Workflows
- Activities
- Workers
- Signals
- Queries
- Child workflows
- Continue-as-new
- Retries
- Visibility and search attributes
- Replay-safe versioning with Cadence APIs

## Supported SDKs

- [x] Go
- [x] Java

## Unsupported In This Skill

The original Temporal-derived content included SDKs and features that are not part of the official Cadence scope for this repository.

See `references/non-compatible/README.md`.
